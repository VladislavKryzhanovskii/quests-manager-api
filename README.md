# quests-manager-api

### Настройка среды 

#### Для запуска приложения необходимо выполнить:
```shell
git clone https://github.com/VladislavKryzhanovskii/quests-manager-api.git
cd quests-manager-api
make build 
make up 
make migrate
```

### Для запуска unit тестов необходимо выполнить:
```shell
make test
```
